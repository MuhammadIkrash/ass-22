# Whole Foods Market - E-Commerce Web App

An interactive and responsive grocery store platform designed to provide users with a seamless shopping experience. Users can browse various categories (Fruits, Meat, Fast Food), search for specific items, and manage their accounts through a secure authentication flow.

## 🚀 Live Demo
(https://ass-22-rosy.vercel.app/)

## ✨ Features
*   **User Authentication:** Fully functional Sign-up and Login pages with validation alerts.
*   **Dynamic Product Display:** A clean, grid-based layout showcasing products with pricing and categories.
*   **Real-time Search:** Filter products instantly using the search bar (e.g., searching for "Apple" or "Mango").
*   **Responsive Design:** Optimized for Desktop, Tablet, and Mobile views using modern CSS techniques.
*   **Interactive UI:** Hover effects, smooth transitions, and user-friendly navigation.

## 🛠️ Tech Stack
*   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
*   **Deployment:** Vercel
